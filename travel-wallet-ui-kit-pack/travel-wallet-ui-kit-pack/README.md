# Travel Wallet POS UI Kit Pack

## What’s inside
- `mockups/01-mockup.png` — target UI (1920×1080)
- `assets/bg-everest.jpg` — background image (1920×1080)
- `assets/icons/*.svg` — logo + action icons
- `theme.css` — CSS variables/tokens (optional)
- `spec.md` — layout + sizing spec
- `App.tsx` — React + TS starter UI

## How to use in Cursor
1) Drop this folder into your project (or unzip into your repo).
2) Ensure your dev server can serve `/assets/...` paths (or move assets into `public/assets`).
3) Ask Cursor:
   "Recreate pixel-perfect from mockups/01-mockup.png using spec.md and theme.css. Use the SVG icons and bg-everest.jpg exactly. Do not improvise."
